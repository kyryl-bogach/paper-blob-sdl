Detailed Description

Permite al usuario manejar fichero para guardar y leer su contenido. Este TDA presenta Nodos con un array de caracteres de longitud máxima de 100 y un entero para la puntuación. De este modo, tenemos una array de MAX Nodos que además contiene la longitud actual de la misma (n)
