Comprueba las colisiones entre una Lista y un Punto y devuelve el número de éstas. Para ello comprueba según la altura y anchura de cada elemento la colisión entre dos rectángulos. Si colisionan entonces, en función del parámetro i, se incrementa o decrementa el punto *valor y se dibujará la Imagen dada por parámetro.

Parameters
pLa Lista a comprobar colisiones.
cuadradoPunto con el que vamos a comprobar las colisiones con la Lista.
plargoLa longitud de los elementos de la primera Lista.
paltoLa altura de los elementos de la primera Lista.
qlargoLa longitud del Punto.
qaltoLa altura del Punto.
iValor que controla si el *valor, se incrementará o decrementará. SI i=0 ENTONCES *valor=*valor+1; SI_NO *valor=*valor-1;
*valorEl valor a modificar si existe colisión.
imgLa Imagen a dibujar si existe colisión. (Sirve para hacer un efecto de colisión).
