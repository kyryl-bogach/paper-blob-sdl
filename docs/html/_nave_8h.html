Inserta elementos en la Lista p con dirreción vectorial [Ratón,Posicion(Nave)].En este caso, se insertan varios disparos en forma de cruz. Se construirá el vector unitario para así controlarlo posteriormente con la velocidad. Se toma como parámetros el Largo y Alto del disparo para cuadrar el disparo. Además, únicamente se disparará si el puntero *mousePulsado está a cero, es decir, si el ratón no está pulsado. Funciona como limitador para que solo puedas disparar con un click.
